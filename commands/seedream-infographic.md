# Seedream 信息图提示词生成器

专门为即梦 Seedream 4.5 生成信息图、图表、数据可视化类提示词，参考 awesome-seedream-4.5 精选信息图提示词库。

## 使用方式

```
/jimeng-workflow:seedream-infographic [需求描述]
```

**示例：**
- `/jimeng-workflow:seedream-infographic 微服务架构图，蓝白配色`
- `/jimeng-workflow:seedream-infographic 番茄炒蛋食谱步骤卡`
- `/jimeng-workflow:seedream-infographic 光合作用原理，黑板风格`

---

用户输入：`$ARGUMENTS`

## 信息图类型识别与模板

### 🏗️ 技术架构图
适用：系统架构、网络拓扑、微服务、数据库设计

```
一张[系统名称]的高级架构图，用于[受众]展示。图中使用图标表示[组件列表]，连接线显示[数据流/调用关系]。风格简洁专业，采用[配色方案]。标注清晰，适合技术文档使用。
```

**参考**（来自 awesome-seedream-4.5 No.35）：
> 为利益相关者展示的现代 Web 应用程序的高级系统架构图。该图应使用图标来表示不同的组件，例如用户浏览器、负载均衡器、Web 服务器、数据库和第三方 API。它们之间的连接应显示数据流。风格应简洁、专业，采用蓝白配色。

---

### 📋 流程图
适用：业务流程、开发流程、操作步骤、决策树

```
一个清晰专业的流程图，展示[流程名称]。包含[阶段列表]，用[形状说明]表示不同节点。连接线有清晰的箭头。配色方案[风格]，适合[使用场景]。
```

**参考**（来自 awesome-seedream-4.5 No.49）：
> 一个清晰专业的流程图，展示了软件开发生命周期（SDLC）。该图应包含规划、设计、开发、测试、部署和维护等阶段，并用不同的形状（矩形、菱形表示决策）表示。连接线应有清晰的箭头。配色方案应采用企业级简约风格。

---

### 🍳 食谱/操作指南卡
适用：食谱步骤、使用说明、DIY 教程

```
一张精美[风格]的[主题]指南卡片。上半部分是[成品插画/照片]。下半部分列出[内容]，每项配有小图标。整体风格[氛围描述]，适合[使用场景]。
```

**参考**（来自 awesome-seedream-4.5 No.36）：
> 一张精美插画食谱卡片，教你制作自制意大利面。上半部分是一幅迷人的成品菜肴插画。下半部分列出了食材和步骤，每种食材旁都配有小图标（面粉、鸡蛋、盐）。整体风格温馨质朴，仿佛来自一本手绘食谱书。

---

### 🔬 科学/教育插图
适用：生物解剖、物理原理、化学反应、地理地质

```
一幅[主题]的详细[风格]插图，用于[教育场景]。图中清晰标注[关键部分]，每个部分颜色分明。风格[视觉风格]，[特殊效果描述]。
```

**参考**（来自 awesome-seedream-4.5 No.47）：
> 一幅详细的复古风格黑板插画，解释了人体心脏的解剖结构。这幅画应以逼真的科学风格绘制，使用白色和彩色粉笔。心脏的所有不同部分都应使用优雅的草书清晰标注。黑板的纹理和粉笔灰应清晰可见。

---

### 📊 数据可视化
适用：统计图表、对比分析、趋势展示

```
一份关于[主题]的精美数据可视化。主视觉是[图表类型]，[数据描述]。设计应[视觉目标]，配色[方案]，能有效传达[核心信息]。
```

**参考**（来自 awesome-seedream-4.5 No.42）：
> 一份关于全球森林砍伐报告的精美且信息丰富的数据可视化。主视觉是一张世界地图，各国根据其森林流失率进行着色。设计应该具有冲击力并能引发情感共鸣。

---

### 🗺️ 地图与空间图
适用：等距地图、旅游地图、城市规划

```
一张[地点/区域]的[风格]插画地图。地图突出显示[关键地标/路线]。风格[视觉风格]，配有[装饰元素]。用于[使用场景]，兼具[功能性]和[艺术性]。
```

---

### 🧠 思维导图
适用：概念梳理、知识结构、创意发散

```
一个生动有机的思维导图，围绕"[核心主题]"展开。中心思想呈[形状]，分支延伸至[子主题列表]。每个分支有独特的颜色，配有小型相关图标。整体风格[视觉风格]。
```

**参考**（来自 awesome-seedream-4.5 No.48）：
> 一个生动有机的思维导图，围绕"创新思维"这一主题展开。中心思想呈云朵状，分支延伸至"头脑风暴"、"正念"、"发散性思维"和"协作"等子主题。每个分支应有独特的颜色，并配有一个小型相关图标。

---
