# Seedance 电影镜头提示词生成器

专门为即梦 Seedance 2.0 生成电影感镜头、多镜头故事板、蒙太奇类视频提示词，参考 awesome-seedance-2-prompts 精选电影镜头提示词库。

## 使用方式

```
/jimeng-workflow:seedance-cinematic [场景描述]
```

**示例：**
- `/jimeng-workflow:seedance-cinematic 一镜到底，女侦探在废弃旅馆中搜查`
- `/jimeng-workflow:seedance-cinematic 9格分镜，朋友们的聚会瞬间`
- `/jimeng-workflow:seedance-cinematic 快速蒙太奇，世界各地地标，0.1秒切换`

---

用户输入：`$ARGUMENTS`

## 电影镜头子类型

### 🎥 一镜到底（Single Take）

无剪辑的连续镜头，强调流畅感和沉浸感。

**结构**：
```
[场景标题]（单次连续镜头）

镜头从[起点：高空/地面/室内]开始，[第一段运动描述]。
[事件1]，镜头[调整方式]。
[事件2]，[环境互动]。
[高潮时刻]，镜头[最终运动]。
最终[收尾画面]，[情绪落点]。
```

**精选参考**（来自 awesome-seedance-2-prompts）：

> 飞车追逐，穿越悬崖城市（单次连续镜头）
> 镜头从一座雕刻在巨石中的宏伟悬崖城市俯冲而下，追逐着一道沿着狭窄崖边道路疾驰的微弱光线。镜头猛地向前冲刺，然后迅速拉回，紧接着贴近飞车尾部的推进器：热浪翻滚，碎石从崖边飞溅。最后一个弯道，突然归于平静：镜头猛然向外拉伸，展现出城市通向一个无边无际、瀑布滋养的山谷。

---

### 🏗️ 起重机/摇臂镜头

从高处俯冲或升起，强调空间感和史诗感。

**结构**：
```
剪辑1（Xs）：宽广的摇臂镜头，从[角度]拍摄[主体]，[背景描述]。
剪辑2（Xs）：缓慢推近特写，聚焦在[细节]上，[情绪细节]。
剪辑3（Xs）：[下一个镜头]。
```

**精选参考**：

> 剪辑1（2秒）：宽广的摇臂镜头，从这对夫妇身后拍摄，城堡在薄雾笼罩的夜晚中闪耀。风吹动着他们的斗篷。
> 剪辑2（2秒）：缓慢推近特写镜头，聚焦在他们紧握的双手上，细微的颤抖。

---

### ⚡ 快速蒙太奇

超高速剪辑，每段 0.1-0.5 秒，强调节奏感和视觉冲击。

**结构**：
```
超高速剪辑切换，每段剪辑[X]秒。
每个镜头都有不同的构图、角度和场景。

序列：
- [时间点]：[场景描述]
- [时间点]：[场景描述]
- [时间点]：[场景描述]
[风格标签：高质量/动漫/实景]。
```

**精选参考**：

> 以0.1秒的间隔，在世界旅游景点和世界遗产的实景画面之间进行超高速的公制蒙太奇切换。序列从东京塔开始，在晴空塔结束，中间的地点是随机的，由系统决定。每个镜头都必须是视频，而不是静态图像，并包含运镜。

> 一段充满各种情人节快乐剧情的视频。超高速剪辑切换，每段剪辑最快0.2秒。每个剪辑都有不同的构图、角度和场景。高质量的日本动漫。先进的灯光和阴影。令人印象深刻且充满活力的摄像机运动。

---

### 🎞️ 多镜头故事板

按叙事逻辑分镜，用 `[切]` 标记切换点。

**结构**：
```
[场景1：铺垫，平稳镜头]。
[切] [场景2：紧张感上升，轻微手持]。
[切] [场景3：高潮，画面晃动]。
[切] [场景4：收尾，情绪落点]。
保持[角色/色调/风格]一致。
```

**精选参考**：

> 根据9格分镜板网格，创作一段电影般的救援蒙太奇。节奏：缓慢铺垫至英勇高潮。
> 第1-2格：铺垫，平稳
> 第3-5格：紧张感上升，轻微手持拍摄
> 第6-7格：紧迫感，画面晃动，剪辑加速
> 第8-9格：英雄时刻，慢动作

---

### 👁️ 第一人称视角（POV）

代入感极强，观众即主角。

**结构**：
```
第一人称视角。[场景设定]。
[主角动作：手部/移动/反应]。
[环境细节：声音/光线/障碍]。
[情绪高潮：发现/惊吓/对峙]。
```

**精选参考**：

> 第一人称视角视频，夜晚，一间老旧房屋里光线昏暗的废弃房间，1990年代美学风格。观众（主角）跪在一个积满灰尘的木制梳妆台前，一个隐藏的抽屉半开着。镜头中的手伸进去，拿出一盘磨损、没有标签的黑色VHS磁带。

---

### 🌀 动态延时/特殊效果

静态主体 + 动态背景，或特殊时间效果。

**精选参考**：

> 让中心人物完全静止，像雕塑一样凝固。只对周围的背景和人物进行快速移动和强烈的运动模糊处理，在静态主体周围营造出动态的延时效果。摄像机必须保持完全固定。

## 镜头语言速查

| 效果 | 描述词 |
|------|--------|
| 史诗感 | 起重机镜头、宽广摇臂、俯冲 |
| 紧张感 | 手持抖动、快速切换、推镜 |
| 沉浸感 | 一镜到底、第一人称、跟踪 |
| 情绪感 | 慢动作、特写、浅景深 |
| 节奏感 | 0.1秒切换、节拍蒙太奇 |
