# Seedance 动作/打斗场景提示词生成器

专门为即梦 Seedance 2.0 生成动作打斗、追逐、对决类视频提示词，参考 awesome-seedance-2-prompts 精选动作类提示词库。

## 使用方式

```
/jimeng-workflow:seedance-action [场景描述]
```

**示例：**
- `/jimeng-workflow:seedance-action 女忍者在雨夜小巷中对战多名敌人`
- `/jimeng-workflow:seedance-action 赛博朋克刺客被包围，电影级CGI`
- `/jimeng-workflow:seedance-action @image1 角色参考，动漫风格打斗`

---

用户输入：`$ARGUMENTS`

## 动作场景子类型

### ⚔️ 武器对决（剑/刀/忍者）

**结构**：
```
[风格标签]。[场景：地点+时间+氛围]。
[角色A描述] 对战 [角色B描述]。
镜头从[起始角度]，[第一个动作]。
[武器碰撞/特效]，镜头[切换/跟踪]。
关键时刻[慢动作]，[情绪高潮]。

```

> 真人版漫画改编·呼吸法终极对决（15秒·超史诗级特效版）
> 核心亮点：水之呼吸（蓝色水龙）VS 雷之呼吸（金色闪电），真人高速对决。
> 风格：好莱坞真人漫画电影感，暗黑武士美学，4K超清，超快速剪辑。

## 通用增强词汇

**镜头**：多角度切换、低角度仰拍、跟踪镜头、手持抖动、慢动作特写
**特效**：火花四溅、碎片横飞、能量爆发、运动模糊、冲击波
**氛围**：戏剧性灯光、雨夜、霓虹、烟雾弥漫、逆光剪影
**风格**：电影级写实、CGI、真人漫改、动漫赛璐珞、4K超清
