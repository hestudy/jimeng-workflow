# Seedream 4.5 图片提示词生成器

为即梦 Seedream 4.5 生成专业图片提示词。

## 使用方式

```
/jimeng-workflow:seedream-prompts [需求描述]
```

**示例：**
- `/jimeng-workflow:seedream-prompts 一张科技公司招聘海报`
- `/jimeng-workflow:seedream-prompts 手冲咖啡步骤信息图`
- `/jimeng-workflow:seedream-prompts 奢华香水产品摄影`

---

## 执行流程

用户输入：`$ARGUMENTS`

### 第一步：分析需求

识别以下信息：
- **类型**：海报 / 信息图 / 摄影 / 插画 / 其他
- **用途**：商业 / 社交媒体 / 教育 / 个人创作
- **风格偏好**：极简 / 复古 / 现代 / 迷幻 / 写实 / 插画
- **宽高比**：根据用途推荐

如果需求不够清晰，先询问 1-2 个关键问题再生成。

### 第二步：生成提示词

参考以下高质量提示词模式：

**海报类模式**（来自 awesome-seedream-4.5 精选）：
> 一张[主题]的[风格]海报。设计采用[视觉特征]。主视觉是[核心元素]。[文字排版说明]。[情感/氛围目标]。--ar [比例]

**信息图类模式**：
> 一张[类型]，展示[内容主题]。图表应包含[具体元素]。风格应[视觉风格]，[颜色方案]。[标注要求]。

**摄影类模式**：
> [主体描述]，[场景背景]。[光线描述]。[构图视角]。[风格氛围]。--ar [比例]

**插画类模式**：
> [插画风格][主题]。[角色/元素描述]。[色彩方案]。[艺术风格参考]。[细节要求]。

### 第三步：输出结果

输出格式：

```markdown
## ✨ 提示词

### 主版本
\`\`\`
[提示词]
\`\`\`

### 变体 A — [风格名称]
\`\`\`
[提示词]
\`\`\`

### 变体 B — [风格名称]
\`\`\`
[提示词]
\`\`\`

## 📐 参数建议
- 宽高比：`--ar X:Y`
- 适用场景：[说明]

## 💡 调整方向
- 想要更[方向A]：[修改建议]
- 想要更[方向B]：[修改建议]
```
