# Seedance 2.0 视频提示词生成器

为即梦 Seedance 2.0 生成专业视频提示词，支持 T2V / I2V / V2V 三种模态。

## 使用方式

```
/jimeng-workflow:seedance-prompts [需求描述]
```

**示例：**
- `/jimeng-workflow:seedance-prompts 两个忍者在竹林中对决`
- `/jimeng-workflow:seedance-prompts 跑车变形成机器人`
- `/jimeng-workflow:seedance-prompts 产品广告，护肤品，温泉场景`
- `/jimeng-workflow:seedance-prompts @image1 角色参考，让她跳舞`

---

## 执行流程

用户输入：`$ARGUMENTS`

### 第一步：分析需求

识别以下信息：
- **类型**：动作打斗 / 科幻奇幻 / 变形变身 / 电影镜头 / UGC生活 / 快速蒙太奇 / 产品广告 / 多镜头故事板
- **模态**：T2V（纯文字）/ I2V（有参考图，用@image）/ V2V（有参考视频，用@video）
- **时长**：4-15秒，根据内容复杂度推荐
- **比例**：16:9横版 / 9:16竖版 / 1:1正方形

如果输入包含 `@image` 或 `@video`，自动识别为 I2V / V2V 模态。

### 第二步：生成提示词

根据类型应用对应结构：

**动作/打斗**（多镜头 + 慢动作高潮）：
> [风格标签]。[场景]。[角色]。镜头从[起点]，[运动]。[动作序列]，关键时刻慢动作。

**科幻/奇幻**（规模感 + 特效）：
> [世界观]。[核心奇观]。镜头[运动]展现规模。[粒子/光效/爆炸特效]。

**变形/变身**（过程描述 + 跟踪镜头）：
> [初始状态]。[变形过程：部件分离/旋转/重组]，[特效]。最终变为[结果]。

**一镜到底**（连续镜头 + 时间轴）：
> [场景]（单次连续镜头）\n镜头从[起点]，[第一段]。[事件]，镜头[调整]。最终[收尾]。

**UGC/生活**（真实感 + 拍摄风格）：
> [场景]，[人物]。[拍摄风格：iPhone/手持/低画质]。[动作]，[细节：模糊/颗粒/构图]。

**快速蒙太奇**（时间点序列）：
> 超高速剪辑，每段[X]秒。序列：[时间点]:[场景1] → [时间点]:[场景2] → ...

**产品广告**（多镜头分段）：
> 镜头1（Xs）：[特写/展示]。镜头2（Xs）：[使用场景]。镜头3（Xs）：[收尾]。

```markdown
**建议时长**：X 秒

### 主版本
\`\`\`
[提示词]
\`\`\`

### 变体 A — [风格]
\`\`\`
[提示词]
\`\`\`

### 变体 B — [风格]
\`\`\`
[提示词]
\`\`\`

[解释关键镜头决策]

## 💡 调整方向
- 想要更强的冲击感：[建议]
- 想要更真实的质感：[建议]
```
