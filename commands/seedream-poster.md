# Seedream 海报提示词生成器

专门为即梦 Seedream 4.5 生成各类海报图片提示词，参考 awesome-seedream-4.5 精选海报提示词库。

## 使用方式

```
/jimeng-workflow:seedream-poster [海报需求]
```

**示例：**
- `/jimeng-workflow:seedream-poster 夏季音乐节，迷幻风格`
- `/jimeng-workflow:seedream-poster 新中式茶饮品牌宣传`
- `/jimeng-workflow:seedream-poster 环保公益，海洋塑料污染`

---

用户输入：`$ARGUMENTS`

## 海报类型识别与模板

根据需求自动匹配以下类型：

### 🎵 活动/演出海报
核心要素：活动名称、视觉主题、氛围、时间地点排版
```
一张[活动名称]的[风格]宣传海报。设计以[核心视觉]为主体，[色彩描述]。[字体/排版风格]。整体氛围[情感目标]。--ar 2:3
```

### 🛍️ 商业促销海报
核心要素：产品/品牌、促销信息突出、视觉冲击力
```
一张[品牌/产品]的[促销主题]海报。设计采用[视觉风格]，主视觉是[产品/模特描述]。"[核心文案]"是海报中最突出的元素。整体营造[情感]。--ar 3:4
```

### 🌍 公益/社会议题海报
核心要素：强烈视觉隐喻、情感冲击、清晰信息传达
```
一张强有力且发人深省的公益海报，主题是[议题]。画面中[超现实/隐喻视觉描述]。底部标语："[文案]"。视觉效果应[情感目标]。--ar 3:4
```

### 🎬 电影/书籍封面
核心要素：类型氛围、主角/核心意象、字体风格
```
一张[类型]风格的[作品名]封面。封面描绘[核心视觉场景]。[光线/色调描述]。标题采用[字体风格]。整体氛围[类型特征]。--ar 2:3
```

### 🏃 体育/健康活动海报
核心要素：动感、活力、参与感
```
一张充满活力、鼓舞人心的[活动名称]海报。设计中包含[动感视觉]，[色彩/形状描述]。活动详情清晰易读。海报应[情感目标]。--ar 3:4
```

## 精选参考提示词（来自 awesome-seedream-4.5）

以下是经过验证的高质量海报提示词，可作为风格参考：

**极简戏剧海报**：
> 一幅莎士比亚《哈姆雷特》演出的极简主义戏剧海报。设计以一个强有力的单一图像为特色：一个戴着金色王冠的人类头骨，置于纯黑背景之上。标题"HAMLET"采用经典、锐利的衬线字体，字母略显腐朽。--ar 2:3

**迷幻音乐节**：
> 一张充满活力、迷幻风格的夏季音乐节海报，名为"Ethereal Bloom"。设计是色彩的超现实爆发，巨大的发光蘑菇、异想天开的奇幻生物和抽象的几何图案交织在一起。音乐节阵容以波浪形、扭曲的字体融入到艺术作品中。--ar 2:3

**公益环保**：
> 一张强有力且发人深省的公益海报，宣传海洋塑料污染问题。画面中，一只海龟在由塑料袋和塑料瓶组成的"海洋"中游弋，呈现出一种超现实且悲伤的景象。海报底部写着标语："这就是你想要的海洋吗？"--ar 3:4
