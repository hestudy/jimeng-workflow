## 与图片提示词的核心差异

视频提示词必须描述**时间维度**：
- 事件如何随时间展开
- 镜头如何运动
- 场景如何切换
- 节奏和情绪如何变化

## 提示词类型识别

### 🥊 动作/打斗类
关键词：打斗、战斗、追逐、对决、忍者、格斗

**模板**：
```
[风格标签]。[场景设定]。[角色描述]。
镜头从[起始位置]开始，[镜头运动]。[动作描述]，[特效描述]。
在关键时刻[慢动作/快切]，[情绪高潮]。
```

**参考**（来自 awesome-seedance-2-prompts）：
> 在竹林中，白忍者和黑忍者之间展开了一场史诗般的电影级大战。白忍者击败了黑忍者。提供一个多镜头动作序列，在关键动作时刻使用慢动作。

---

### 🚀 科幻/奇幻类
关键词：科幻、机甲、龙、宇宙、赛博朋克、末日

**模板**：
```
[世界观设定]。[核心视觉奇观描述]。
镜头[运动方式]，展现[规模感/震撼感]。
[特效描述：粒子/光效/爆炸]。[情绪收尾]。
```

**参考**：
> 一架巨大的装甲机甲出现在一座现代化城市的中央，它雷鸣般的脚步声震撼着街道，冲击波穿透高耸的建筑。尘土和碎片弥漫在空气中，受惊的人群惊慌失措地奔跑。

---

### 🔄 变形/变身类
关键词：变形、变身、变换、进化、转变

**模板**：
```
[初始状态描述]。
[变形过程]：[部件/元素]平滑地[分离/旋转/重组]，[特效]。
最终变为[结果状态]。镜头[跟踪/环绕]记录全程。
```

**参考**：
> 一段10秒的动态视频展示了一辆现代跑车在原地变形。金属车身部件平滑地分离、旋转并重新组装，变形为一个高大帅气的人形机甲。视觉特效包括金属光泽、机械齿轮咬合声和蓝色能量光芒。

---

### 🎬 电影镜头类
关键词：一镜到底、起重机镜头、跟踪镜头、电影感

**模板（一镜到底）**：
```
[场景设定]（单次连续镜头）

镜头从[起点]开始，[第一段运动]。
[事件1]，镜头[调整]。
[事件2]，镜头[继续运动]。
最终[收尾镜头]，[情绪落点]。
```

**参考**：
> 飞车追逐，穿越悬崖城市（单次连续镜头）
> 镜头从一座雕刻在巨石中的宏伟悬崖城市俯冲而下，追逐着一道沿着狭窄崖边道路疾驰的微弱光线。镜头猛地向前冲刺，然后迅速拉回，紧接着贴近飞车尾部的推进器。

---

### 📱 UGC/生活类
关键词：自拍、日常、vlog、随手拍、生活感

**模板**：
```
[场景]，[人物描述]。
[拍摄风格：iPhone随手拍/手持/低画质]。
[动作/表情描述]，[细节：运动模糊/颗粒感/构图不佳]。
```

**参考**：
> 卧室场景照片，卧室略显凌乱但充满少女气息。这是一张用普通iPhone拍摄的照片。图像有轻微的运动模糊，角度笨拙，构图不佳，随手抓拍风格，图像颗粒感明显，日常Vlog。

---

### ⚡ 快速蒙太奇类
关键词：快速切换、蒙太奇、高速剪辑、多场景

**模板**：
```
超高速剪辑切换，每段剪辑[X]秒。
序列：
- [时间点]：[场景1]
- [时间点]：[场景2]
- [时间点]：[场景3]
每个镜头都有不同的构图、角度和场景。
```

**参考**：
> 以0.1秒的间隔，在世界旅游景点和世界遗产的实景画面之间进行超高速的公制蒙太奇切换。序列从东京塔开始，在晴空塔结束，中间的地点是随机的。

---

### 📦 产品广告类
关键词：广告、产品、展示、商业

**模板**：
```
镜头1（X秒）：[特写/产品展示]，[光线/质感描述]。
镜头2（X秒）：[使用场景/效果展示]，[特效]。
镜头3（X秒）：[品牌/收尾]，[动态图形]。
```

**参考**：
> 镜头1：特写。镜头缓慢推近。一名男子在实验室里检查一个绿色瓶子。
> 镜头2：微距对焦。一股强烈的冷蒸汽冲击瓶子，瓶身结霜。
> 镜头3：超特写。瓶子周围的数字全息图发出光芒，显示温度。

---

### 🎭 多镜头故事板类
关键词：故事板、分镜、多场景、叙事

**模板**：
```
[场景1描述]。
[切] [场景2描述]。
[切] [场景3描述]。
保持[角色/风格/色调]一致。
```
