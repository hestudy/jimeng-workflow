## 核心框架：五要素导演法

所有好的 AI 视频提示词，本质上都在回答五个问题：

**实战公式**：
```
[角色指定] + [场景设定] + [动作/剧情描述] + [镜头语言] + [氛围/声音]
```

> 不需要每次都写满五个要素，但要素越完整，生成结果越接近脑海中的画面。
> 像"导演手记"一样写，不像"许愿"一样写。

## 九大创作动机分类

### 一、完整故事短片

**类型A：生活情景剧**（日常场景 + 细腻微动作链 + 情绪转折弧线）
```
女孩在优雅的晒衣服，晒完接着在桶里拿出另一件，用力抖一抖衣服。
```

**类型B：电影感叙事**（高冲突 + 强镜头语言 + 类型片调性）
```
谍战片风格，@图片1作为首帧，镜头正面跟拍穿着红风衣的女特工向前走，
不断有路人遮挡，走到拐角处消失，一个戴面具的女孩在拐角处躲着盯着她...
全程不要切镜头，一镜到底。
```

**类型C：搞笑/创意反转**（颠覆常理 + 荒诞设定 + 方言/跨次元）
```
画里面的人物心虚的表情，眼睛左右看了看探出画框，快速将手伸出画框
拿起可乐喝了一口，然后露出一脸满足的表情，这时传来脚步声...
```

---

### 二、产品展示/商业广告

**类型A：产品特写**（多图拼合细节 + 专业运镜参考）
```
对@图片2的包包进行商业化摄像展示，包包侧面参考@图片1，
表面材质参考@图片3，要求将包包细节均有所展示，背景音恢宏大气。
```

**类型B：时间轴广告脚本**（精确到每秒的节奏控制）
```
0-2秒：快速四格闪切，四款产品依次定格，特写材质光泽与品牌字样。
3-6秒：特写磁吸扣"咔哒"吸合，展示质感与便捷性。
7-12秒：快速切换佩戴场景，通勤/休闲/出街。
13-15秒：产品并排陈列，品牌名弹出。
```

**类型C：对比手法广告**（视觉冲击 + 问题/解决方案结构）
```
女人在优雅的做饭，没有烟雾，镜头快速向右边摇动，
拍摄男人满头大汗面红耳赤在做饭，浓烟滚滚，
镜头向左边摇动推进拍摄桌面上的一个油烟机，油烟机在疯狂抽烟。
```

### 三、复刻/模仿爆款视频

最简指令：
```
参考@视频1的运镜+画面切换节奏，用@图片1的[角色]进行复刻。
```

进阶：分别指定素材角色——运镜参考视频A，动作参考视频B，角色用图片C。

"换人不换动作"是最高频用法：把别人视频里的角色换成你的，动作和镜头保持不变。

---

### 四、特效/转场/视觉奇观

```
以黑幕开场，参考视频1的粒子特效和材质，金色鎏金材质的沙砾从画面
左边飘出并向右覆盖，@图片1的字体逐渐出现在画面中心。
```

转场核心公式：**从什么画面 → 经过什么效果 → 变成什么画面**

风格关键词一句话生效：`黑白水墨风格` `赛博朋克` `科幻数据空间` `希区柯克变焦`

---

### 五、音乐卡点/MV

```
@图片1-6的风光场景图，参考@视频中的画面节奏，
转场间画面风格及音乐节奏进行卡点。
```

多图 + 一段音乐视频 = 自动卡点MV，加上`画面张力强`、`更梦幻`、`光影变化`提升质感。

---

### 六、视频延长/编辑

```
延长15s，参考@图片1的形象，补充一段脑洞广告。
画面1：侧面固定镜头，[角色]冲出栅栏...
画面2：[角色]在沙地盘旋...
画面3：背景是雪山镜头，[角色]从山坡飞越过...
```

延长时写清新增部分的具体内容，不要只说"延长5秒"。
注意：延长5秒，生成时长也要选5秒。

---

### 七、角色说话/唱歌/表演

对话格式：
```
角色名（动作描述）："台词内容"
喵酱（舔毛翻眼）："家人们谁懂啊..."
旺仔（歪头晃尾巴）："你还好意思说我？..."
```

可指定语言：四川话、西班牙语、韩语、豫剧腔——直接写就行。
音色参考：`语气和音色参考@视频1`

---

### 八、分镜/脚本生成视频

```
将@图1以从左到右从上到下的顺序进行漫画演绎，
保持人物说的台词与图片上的一致，分镜切换加入特殊音效，
整体风格诙谐幽默；演绎方式参考@视频1。
```

有编导基础的用法：直接画分镜 + 写脚本，AI 精准执行。

---

### 九、一镜到底沉浸体验

```
@图片1-5，一镜到底的追踪镜头，从街头跟随跑步者上楼梯、
穿过走廊、进入屋顶，最终俯瞰城市。
```

多张图按顺序 = 场景路径，模型自动做场景间的过渡。
速度变化写清楚：`越来越快` `缓缓飘至` ——节奏感来自速度描述。

## 六大黄金法则

**法则一：素材分工要"点名到人"**
- ❌ 上传了几张图和一段视频，生成一段打斗视频
- ✅ @图1@图2为长枪角色，@图3@图4为双刀角色，模仿@视频1的动作，在@图5的枫叶林中打斗

**法则二：动作要拆成"微动作链"**
- ❌ 女生在做饭
- ✅ 女人在优雅的做饭，没有烟雾，镜头快速向右边摇动，拍摄男人满头大汗面红耳赤在做饭，浓烟滚滚

**法则三：镜头语言决定"质感段位"**

| 术语 | 效果 |
|------|------|
| 推 | 镜头从远到近，聚焦某物 |
| 拉 | 镜头从近到远，展示全景 |
| 摇 | 镜头左右旋转，扫视环境 |
| 移 | 镜头整体平移，跟随角色 |
| 环绕 | 围绕角色旋转，强调主体 |
| 俯拍 | 从上往下，上帝视角 |
| 仰拍 | 从下往上，英雄感/压迫感 |
| 一镜到底 | 不切镜头连续拍，沉浸感 |
| 希区柯克变焦 | 主体不变背景突然拉近，眩晕/惊恐感 |

**法则四：时间轴脚本是精确控制的终极武器**
```
0-3秒：[画面1的描述]
3-6秒：[画面2的描述]
6-10秒：[画面3的描述]
10-15秒：[画面4的描述+结尾处理]
```

**法则五：类型片调性先定好**
开头写"谍战片风格"/"恐怖悬疑"/"商业广告"，给模型整体方向。

**法则六：对话/台词直接写进提示词**
用引号写出台词，模型会按照你写的台词生成对应语音。
