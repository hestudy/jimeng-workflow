- 迷幻风格：`一张充满活力、迷幻风格的夏季音乐节海报。设计是色彩的超现实爆发，巨大的发光蘑菇、异想天开的奇幻生物和抽象的几何图案交织在一起。--ar 2:3`

| 用途 | 比例 |
|------|------|
| 竖版海报/书籍封面 | `--ar 2:3` |
| 标准竖版 | `--ar 3:4` |
| Instagram 竖版 | `--ar 4:5` |

## 输出规范

每次生成提示词时，输出：

1. **主提示词**（中文版）
2. **英文版**（可选，部分场景英文效果更好）
3. **2个风格变体**
4. **使用建议**（宽高比、调整方向）

始终保持提示词具体、可执行，避免模糊描述。
