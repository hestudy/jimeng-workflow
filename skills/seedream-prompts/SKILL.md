## 通用规则（所有场景适用）

**规则 1：用自然语言清晰描述画面**
- 结构：**主体 + 行为 + 环境**，可补充风格、色彩、光影、构图等美学元素
- ✅ 正确：`一个穿着华丽服装的女孩，撑着遮阳伞走在林荫道上，莫奈油画风格。`
- ❌ 避免：`一个女孩，撑伞，林荫街道，油画般的细腻笔触。`

**规则 2：明确应用场景和用途**
- 有明确用途时，在提示词中写明图像类型
- ✅ 正确：`设计一个游戏公司的 logo，主体是一只在用游戏手柄打游戏的狗，logo 上写有公司名 "PITBULL"。`

**规则 3：提升风格渲染效果**
- 使用精准的风格词（如：儿童绘本、赛博朋克、莫奈油画）
- 有明确风格需求时，建议提供参考图像

**规则 4：提高文本渲染准确度**
- 将要生成的文字内容放在**双引号**中
- ✅ 正确：`生成一张海报，标题为 "Seedream 4.5"`
- ❌ 避免：`生成一张海报，标题为 Seedream 4.5`

**规则 5：明确图片编辑目标**
- 使用简洁明确的指令，说明需要修改的对象及具体操作，避免模糊代词
- 如需保持其他内容不变，在提示词中明确强调
- ✅ 正确：`让图中最高的那只熊猫穿上粉色的京剧服饰并戴上头饰，并保持动作不变。`
- ❌ 避免：`让它穿上粉色衣服。`

## 任务类型识别与模板

根据用户需求，自动识别以下任务类型：

### 🖼️ 文生图

适用：从零开始生成图像，包括海报、插画、信息图、产品图、人物图等。

**核心原则**：Seedream 4.5 理解能力强，**简洁精确的提示优于堆叠华丽词汇**。

**模板结构**：
```
[主体描述] + [行为/状态] + [环境/背景] + [风格/氛围] + [光线/色彩] + [构图/视角] + [文字要求（双引号）] + [--ar 宽高比]
```

**参考案例**：
- 场景图：`一个凌乱的办公桌桌面。桌面上有一台开着的笔记本电脑，屏幕显示绿色代码；旁边一个马克杯，杯上写着 "Developer"，杯口冒出热气；一本摊开的书，页面是维恩图；一支钢笔，笔帽掉在旁边。背景是模糊的书架。阳光从右侧照射，在桌上形成光影。`
- 知识图解：`在黑板上画出下列二元一次方程组及其相应的解法步骤：5x + 2y = 26；2x - y = 5。`
- 信息图：`绘制一张信息图，展示通货膨胀的成因，每条成因独立呈现，并配有简洁图标。`
- 海报：`一幅莎士比亚《哈姆雷特》演出的极简主义戏剧海报。设计以一个强有力的单一图像为特色：一个戴着金色王冠的人类头骨，置于纯黑背景之上。标题 "HAMLET" 采用经典、锐利的衬线字体。--ar 2:3`

### ✏️ 图生图 — 图像编辑

适用：对已有图像进行增加、删除、替换、修改等操作。

**核心原则**：
- 使用简洁明确的文字，准确指示需要编辑的对象与变化要求
- 画面复杂时，可用**箭头、线框、涂鸦**等视觉信号指明编辑区域

**参考案例**：
- 增加：`给图中女生增加相同款式的银色耳线和项链。`
- 删除：`去掉女生的帽子。`
- 替换：`把最大的面包人换成牛角包形象，保持动作和表情不变。`
- 修改：`让图中三个机器人变成透明水晶材质，颜色从左到右分别变成红黄绿。并让绿色的机器人跑起来，黄色的走路，红色的站立。`
- 涂鸦指定区域：`将房间内红色涂抹位置放入电视，蓝色涂抹位置放入沙发，不改变其他布局，确保放入物体和整张图的原木风格一致。`

---

### 🎨 图生图 — 参考图生图

适用：从参考图中提取人物形象、艺术风格、产品特征，完成角色创作、风格迁移、产品设计等。

**核心原则**：在提示词中明确两部分：
1. **指明参考对象**：希望从参考图中提取并保留的元素
2. **描述生成画面**：希望生成的画面内容、场景等细节

**参考案例**：
- 参考人物形象：`参考图中的人物形象，做一个动漫人物手办，放在桌上，后面放置一个印有角色图像的生日礼物包装盒。`
- 参考风格：`参考图标的线性简约风格，设计9个不同场景下的应用 icon，包括：音乐、天气、日历、相机、聊天、地图、闹钟、购物车、笔记本，保持一致的配色。`
- 参考款式：`生成四件不同材质和颜色的上衣，衣服款式与图中女生身上的衣服一致，只要衣服，不要模特。`

---

### 📐 图生图 — 草图转效果图

适用：将平面图、线稿图、手绘原型转化为高保真效果图。

**核心原则**：
- 提供清晰的原始图片，若图中有文字说明，在提示词中注明"遵循图中文字内容进行生成"
- 明确主体与要求（如：高保真 UI 界面、现代简约风格客厅实景图）
- 明确指出需与参考图保持一致的关键要求（如：家具位置与参考图一致）

**参考案例**：
- 平面图转实景：`根据这张平面图，生成"现代简约风精装客厅 + 开放式餐厅"的实景图（房间布局、家具位置完全匹配例图）。地中海风格配色，空间结构和方向始终与例图一致。`
- 手绘原型转 UI：`这是一个 Web 端房屋租赁软件的详情页手绘图，请根据图中文字示意，将其渲染成高保真的 UI 界面，并在画廊区域增加一些示例图片。`

---

### 🖼️🖼️ 多图输入

适用：同时输入多张图像，完成替换、组合、迁移等复合编辑操作。

**核心原则**：在提示词中清楚指明不同图像需要编辑/参考的对象及操作。

**参考案例**：
- 替换：`将图一的主体替换为图二的主体。`
- 组合：`让图一人物穿上图二的服装。`
- 风格迁移：`参考图二的风格，对图一进行风格转换。`

---

### 📚 多图输出

适用：生成角色连贯、风格统一的图像序列，适用于分镜、漫画、IP 产品、表情包等场景。

**触发词**：`一系列`、`一套`、`组图`，或具体数字（如"生成四张图"）。

**参考案例**：
- 品牌视觉：`参考这个 LOGO，做一套户外运动品牌视觉设计，品牌名称为 "GREEN"，包括包装袋、帽子、卡片、手环、纸盒、挂绳等。绿色视觉主色调，简约现代风格。`
- 影视分镜：`生成四张图，影视分镜，分别对应：宇航员在空间站维修飞船、突然遇到陨石带袭击、宇航员紧急躲避、受伤后惊险逃回飞船。`
- 系列壁纸：`生成周一到周日共七张手机壁纸图，自然景观，每张图对应标注日期。`

## 宽高比速查

| 用途 | 比例 |
|------|------|
| 竖版海报 / 书籍封面 | `--ar 2:3` |
| 标准竖版 | `--ar 3:4` |
| 正方形 | `--ar 1:1` |
| 横版 / 视频缩略图 | `--ar 16:9` |
| 横版摄影 | `--ar 3:2` |

## 输出规范

每次生成提示词时，输出：

1. **主提示词**（中文版）
2. **英文版**（可选，部分场景英文效果更好）
3. **2 个风格变体**
4. **使用建议**（宽高比、调整方向）

始终保持提示词具体、可执行，避免模糊描述。
