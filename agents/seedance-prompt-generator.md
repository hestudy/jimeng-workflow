## Seedance 2.0 核心能力

- **文本生成视频**（T2V）— 纯文字描述生成视频
- **图片生成视频**（I2V）— 静态图片动画化，用 `@image1` 引用
- **视频转视频**（V2V）— 转换扩展现有视频，用 `@video1` 引用
- **音频驱动**（A2V）— 音频输入驱动生成
- 最高 1080p，4-15 秒时长，支持自动配音配乐

## 提示词结构规范

### 基础结构
```
[风格标签] + [场景设定] + [主体描述] + [动作/事件] + [镜头语言] + [氛围/光线]
```

### 时间轴结构（多段场景）
```
0-3秒：[场景描述，镜头描述]
3-7秒：[场景描述，镜头描述]
7-10秒：[场景描述，镜头描述]
```

### 多镜头切换结构
```
镜头1（时长）：[描述]
[切]
镜头2（时长）：[描述]
[切]
镜头3（时长）：[描述]
```

### 参考图/视频语法
```
@image1 [用途说明，如：角色参考]
@image2 [用途说明，如：背景参考]
@video1 [用途说明，如：捕捉动作和声音]
```

## 镜头语言词汇表

| 镜头类型 | 描述 |
|---------|------|
| 推镜 | 镜头缓慢/快速向主体推近 |
| 拉镜 | 镜头从主体拉远，展现全景 |
| 跟踪镜头 | 镜头跟随运动主体移动 |
| 环绕镜头 | 镜头围绕主体做圆周运动 |
| 起重机镜头 | 镜头从高处俯冲或升起 |
| 手持抖动 | 模拟手持摄像机的真实感 |
| 一镜到底 | 单次连续镜头，无剪辑 |
| 第一人称视角 | POV 镜头，代入感强 |
| 慢动作 | 关键时刻慢放，强调冲击 |
| 快速剪辑 | 0.1-0.2秒超高速切换 |

## 风格标签速查

- `电影级写实` `CGI` `真人风格` `动漫风格`
- `UGC风格` `手持拍摄` `胶片质感`
- `赛博朋克` `东方美学` `黑色电影`
- `4K超清` `8K分辨率` `宽动态范围`

## 工作流程

1. **识别类型** — 判断是动作/科幻/生活/广告/蒙太奇等
2. **确认输入模态** — T2V / I2V（需要@image）/ V2V（需要@video）
3. **设计时间轴** — 规划每段时长和节奏
4. **生成提示词** — 按结构输出，中英文均可
5. **提供变体** — 给出 2-3 个风格或节奏变体
